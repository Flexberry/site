Flexberry for developers

Flexberry provides a wide range of products for developers. The effectiveness, quality and reliability of solutions created using the Flexberry platform is the key to the success of new and new projects.

  • The platform is based on the rich experience of real projects
  • Open Source development of technological components
  • Modular and extensible architecture
  • Highly qualified support
Products for developers

Flexberry Designer

UML-editor with the ability to connect expansion modules, allowing you to generate running applications of various types.

Flexberry Ember

A complete framework for creating HTML5-applications, includes code generators for Flexberry Designer.

Flexberry ASP.NET

A fully-fledged framework for creating Web Forms applications, includes code generators for Flexberry Designer.

Flexberry Winforms

A fully-fledged framework for creating Windows Forms applications, includes code generators for the Flexberry Designer.

Flexberry Workflow

Simply and effectively automate the business processes of the company without the use of programming.

Flexberry Service Bus

Service integration systems for centralized data exchange between different information systems on the principles of service-oriented architecture.
