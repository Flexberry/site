Flexberry для разработчиков

Flexberry предоставляет широкий ряд продуктов для разработчиков. Эффективность, качество и надёжность решений, созданных с использованием платформы Flexberry - залог успешности новых и новых проектов.

  • Платформа базируется на богатом опыте реальных проектов
  • Open Source разработка технологических компонент
  • Модульная и расширяемая архитектура
  • Высококвалифицированная поддержка
Решения для разработчиков

Flexberry Designer

UML-редактор с возможностью подключения модулей расширения, позволяющих сгенерировать работающие приложения различных типов.

Flexberry Ember

Полноценный фреймворк для создания HTML5-приложений, включает в себя генераторы кода для Flexberry Designer.

Flexberry ASP.NET

Полноценный фреймворк для создания Web Forms-приложений, включает в себя генераторы кода для Flexberry Designer.

Flexberry Winforms

Полноценный фреймворк для создания Windows Forms-приложений, включает в себя генераторы кода для Flexberry Designer.

Flexberry Workflow

Просто и эффективно автоматизируйте бизнес-процессы компании без применения программирования.

Flexberry Service Bus

Сервис интеграции систем для централизованного обмена данными между различными информационными системами на принципах сервис-ориентированной архитектуры.

Flexberry включена в реестр отечественного ПО

Программная платформа Flexberry включена в реестр отечественного ПО. Платформа содержит ряд интересных возможностей для отечественных и иностранных компаний-разработчиков ПО. Познакомьтесь с перечнем функциональных возможностей и смело выбирайте платформу Flexberry для своих проектов.

Инструкция по установке

Продукты платформы Flexberry могут быть скачаны с данного сайта и установлены для ознакомления и последующего приобретения лицензий. Для обеспечения комфортного процесса установки продуктов реализован специальный установщик, автоматически получающий лицензии через Интернет, а также подготовлена инструкция по установке.

Инструкция по тех. сопровождению

Платформа Flexberry предоставляет услуги по поддержке и техническому сопровождению продуктов платформы, что даёт дополнительную гарантию всем пользователям платформы. Для удобства пользователей была разработана инструкция по техническому сопровождению платформы Flexberry.
